# Security Policy

## Supported Versions

Support for Android versions 

| Version | Supported          |
| ------- | ------------------ |
| 12 >   | :white_check_mark: |
| 11.x   | :white_check_mark: |
| < 4.4.x   | :x: |
## Details on unsupported versions 

Bugs that occur in versions prior to Android 11 will not be addressed because it is considered an unsupported version.
